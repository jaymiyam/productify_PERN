# Productify - A Product sharing website

This is a PERN stack practice project from Codesistency's Youtube online course.

## Tech Stack
- Javascript: Frontend
- Typescript: Backend
- PostgresQL (NeonDB & Drizzle ORM): Database
- ExpressJS: Backend Framework
- React: Frontend Framework
- NodeJS: Backend Runtime
- TailwindCSS: Frontend styling
- DaisyUI: Frontend styling
- Clerk: Authentication
